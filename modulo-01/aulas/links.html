Usando Links Externos

Para criar um link utilizamos a tag <a href="https://site.com">Texto que ficará como Hiperlink</a>

Links externos servem para levar para sites diferentes por isso, é recomendável que quando estivermos utilizando links externos adicionarmos dois parâmetros dentro da tag de âncora: <a href="https://site.com" target="_blank" rel="external">Texto que ficará como Hiperlink</a>

  • href = "https://www.link.com" - Cria uma referência hipertexto.
    • hreflang = "en" - Informa para o navegador que o link que será aberto está em outro idioma. (o "en" que foi utilizado indica que está em inglês)
  • target = " " - Informa para o navegador como o link deve ser aberto
    • _blank - Vai abrir o link em uma nova janela em branco.
    • _self - Vai abrir o link na janela ou frame atual.
    • _top - vai desfazer todos os frames e abrir o destino no navegador completo.
    • _parent - similar ao uso do _top em uma referência à janela mãe.
    • nome-do-frame - caso esteja utilizando em frames, indicar o nome da janela a abrir.
  • rel = " " - Indica qual é a natureza do link.
    • next - Indica que o link é para a próxima parte do documento atual.
    • prev - Indica que o link é para a parte anterior do doccumento atual.
    • author - Indica que é um link para o site do autor do artigo atual.
    • external - Indica que é um link para o outro site que não faz parte do site atual.
    • nofollow - Indica que é um link para um site não endossado, como um link pago.

Usando Links Internos

Os links internos servem para fazer uma ligação com outra página que está dentro do nosso servidor e diz respeito a nossa página. Para fazer um link com uma página interna utilizamos a tag de âncora da seguinte forma: <a href="https://site.com" target="_self" >Texto que ficará como Hiperlink</a>

  • target ="_self" - Indica que o link deve ser aberto na própria página

Links para download

Podemos utilizar links para fazer o download de arquivos, para isso utilizamos a tag a e os parametros download e type da seguinte forma:

<a href="livro/meulivro.pdf" download="livro/meulivro.pdf" type="application/pdf">Livro em pdf</a>

<a href="livro/meulivro.zip" download="livro/meulivro.zip" type="application/zip">Livro em zip</a>
