Estilizando

É possível utilizar o elemento style="" dentro de uma tag para mudar algumas propriedades como cor, tamanho, fonte, alinhamento, etc que é o que chamamos de estilos inline. Entretanto, uma caracteerística deste estilo é que ele só é aplicado ao elemento selecionado.
