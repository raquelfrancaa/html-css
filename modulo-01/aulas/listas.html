Trabalhando com Listas

Temos as listas ordenadas que são demarcadas pelas tags <ol>...</ol> e cada ítem da lista deve ser colocado entre uma <li>...</li> sendo que a partir do HTML5 não é mais necessário o uso da tag de fechamento </li>

As listas ordenadas numeram cada item da lista. Exemplo:

  1. Acordar
  2. Tomar Café
  3. Escovar os Dentes

É possível mudar o tipo de enumeramento da lista ordenada da seguinte forma: <ol type = " "> como vamos ver no exemplo a seguir:

Temos o tipo A: <ol type = "A">

  1. Acordar
  2. Tomar Café
  3. Escovar os Dentes

Temos o tipo a: <ol type = "a">

  1. Divida por 25
  2. Tomar Café
  3. Escovar os Dentes

Temos o tipo I: <ol type = "I">

  1. Acordar
  2. Tomar Café
  3. Escovar os Dentes

Temos o tipo i: <ol type = "i">

  1. Acordar
  2. Tomar Café
  3. Escovar os Dentes

Listas Não Ordenadas

As listas não ordenadas são listas que não tem ordem definida, ou seja, não são numeradas. Exemplo:

  • Cortar o cabelo
  • Passear com os cachorros
  • Comprar um sorvete

Também podemos personalizar essas listas, como vamos ver a seguir:

Temos o tipo disc: <ul type = "disc"> (que é o tipo padrão)

  • Cortar o cabelo
  • Passear com os cachorros
  • Comprar um sorvete

Temos o tipo circle: <ul type = "circle">

  • Cortar o cabelo
  • Passear com os cachorros
  • Comprar um sorvete

Temos o tipo square: <ul type = "square">

  • Cortar o cabelo
  • Passear com os cachorros
  • Comprar um sorvete
