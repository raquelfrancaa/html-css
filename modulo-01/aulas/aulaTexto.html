Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando a tag <b> (não semântica)

Nesta frase temos um termo em destaque utilizando a tag strong (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico utilizando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase utilizando a tag em (semântica)

Texto Marcado

Podemos criar também texto marcado com a tag mark

Texto Grande e Texto Pequeno

Estamos criando um texto grande com a tag big (que não deve ser utilizada pois está obsoleta) e um texto pequeno com a tag small (que não está obsoleta) neste parágrafo

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Fazemos isso com a tag <del>

Texto Inserido

Podemos marcar um texto como inserido com a tag ins (semântica) para dar uma ênfase e indicar que ele foi adicionado

Existe também o sublinhado com a tag <u> (não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x2+3 com a tag sup

Texto Subscrito

Para inserir coisas tipo H2O com a tag sub
